


To run Nginx on Ubuntu, 

Install it using sudo apt update
sudo apt install nginx


The service starts automatically, but can be managed with 


	sudo systemctl stop nginx

	sudo systemctl start nginx

	sudo systemctl restart nginx

	sudo systemctl status nginx



Open port 80 via the firewall with 


	sudo ufw app list

	sudo ufw allow 'Nginx HTTP'

	sudo ufw status

🔴 STEP 3: Remove Nginx completely (recommended)

This removes:

* Nginx binaries
* Default configs
* Service files


sudo apt remove --purge nginx nginx-common nginx-core -y


---

🔴 STEP 4: Remove leftover config & cache files

Sometimes configs remain even after purge.


sudo rm -rf /etc/nginx
sudo rm -rf /var/log/nginx
sudo rm -rf /var/cache/nginx


---

🔴 STEP 5: Clean unused packages


sudo apt autoremove -y
sudo apt autoclean


---

🔴 STEP 6: Restore `/var/www` (optional but clean)

If you don’t need web hosting files anymore:


sudo rm -rf /var/www


Or recreate default empty structure:


sudo mkdir -p /var/www
sudo chown -R root:root /var/www


---

🔴 STEP 7: Reset UFW (optional)

You didn’t enable it, but to **fully reset**:


sudo ufw reset


Check:


sudo ufw status


Expected:


Status: inactive


---

🔴 STEP 8: Verify NOTHING is listening on port 80
